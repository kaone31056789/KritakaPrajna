# KritakaPrajna

Desktop AI assistant with multi-provider model access in a single app.

Current installer release: v2.7.0

## Free App Policy

- KritakaPrajna is free to use.
- Public builds are distributed through GitHub Releases.
- End users can download and use the installer without charge.

## GitHub Distribution Policy

- This public repository is release-focused.
- Only release metadata and installer artifacts are intended to be published here.
- Full source code is not published in this repository to prevent direct copy-and-rebrand forks.

## Open-Source Note

- The project remains community-oriented and transparent through release notes and changelogs.
- If source snapshots are published in future, licensing details will be announced in releases.

## Download And Install

1. Open the Releases page:
   https://github.com/kaone31056789/KritakaPrajna/releases
2. Download the installer:
   KritakaPrajna-Setup-2.7.0.exe
3. Run the installer and complete setup.
4. Launch the app and add your provider API key(s).
